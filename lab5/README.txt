Завантажте папку з проектом.

Відкрийте файл Main.java, який містить метод main. Він є точкою входу для запуску програми.

Запустіть Main.java через ваше середовище розробки (наприклад, правою кнопкою миші оберіть "Run").

Перегляньте результати роботи програми.


Після запуску програми у консолі буде виведено:

Загальну споживану потужність увімкнених приладів.

Список приладів, відсортованих за зростанням потужності.

Список приладів, що знаходяться в заданому діапазоні електромагнітного випромінювання (наприклад, від 10 до 40 мкТл).
